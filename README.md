# 🌟 Landing Page - Marcela Saint Martin

Curadoria oficial da Marcela Saint Martin - Landing page completa com 4 seções funcionais.

## 📋 Conteúdo

- ✅ **Seção 1**: Migração da livraria (roxo + computadores)
- ✅ **Seção 2**: Apresentação da Marcela (bege + dropdown)  
- ✅ **Seção 3**: Categorias por idade (bege + cards 2x2)
- ✅ **Seção 4**: Cupom MARCELA (lavanda + botão ação)

## 🚀 Como usar

### **Opção 1: Deploy Rápido (Recomendado)**

1. **Vercel** (Gratuito):
   - Acesse [vercel.com](https://vercel.com)
   - Faça upload da pasta completa
   - Deploy automático!

2. **Netlify** (Gratuito):
   - Acesse [netlify.com](https://netlify.com)
   - Arraste a pasta para o dashboard
   - Publicação instantânea!

### **Opção 2: Desenvolvimento Local**

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm start

# Buildar para produção
npm run build
```
